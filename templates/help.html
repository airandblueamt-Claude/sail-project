How It Works

Booking Equipment

1
Browse

Go to Browse Equipment in the sidebar. Use the filters to find what you need — filter by category or set "Bookable" to see only reservable items.

2
Select an Asset

Click on an equipment model to see its registered assets. Each asset shows its current status — look for ones marked available.

3
Book It

Click the Book button next to an available asset. Pick your dates and explain why you need it — this helps the admin prioritise.

4
Wait for Approval

Your booking goes to pending. An admin will review and approve or reject it. You can track the status in My Bookings.

5
Pick Up & Return

Once approved, the admin will hand over the equipment (checked out). When you're done, return it and the admin marks it as returned.

Submitting a Ticket

1
Create a Ticket

Go to My Tickets and click New Ticket. Choose the type:

  • maintenance — something needs repair
  • incident — something is broken or not working
  • move — equipment needs to be relocated
  • new request — request new equipment
  • decommission — equipment should be retired
2
Track Progress

Your ticket moves through: open → in progress → resolved → closed. You can add comments to provide more details.

Admin Guide

Adding New Equipment

  1. Go to Browse Equipment → click + Add Equipment Model
  2. Fill in the name, brand, category (or create a new one), specs, and quantity
  3. Check Bookable if employees should be able to reserve it
  4. Save → you land on the model detail page
  5. Click + Register Asset to add individual physical units (each gets a unique asset tag like SAIL-0001)
  6. Add serial number and location for each unit

Equipment Model = product type (e.g. "Dell Monitor U2722DE"). Asset = a specific physical unit of that model.

Managing Bookings

  1. Go to Booking Approvals in the sidebar
  2. Approve or Reject pending requests
  3. When the employee comes to collect: click Hand Over to mark as checked out
  4. When they return: click Receive Return to mark as returned and free up the asset

Managing Tickets

  1. Go to All Tickets to see every ticket across the organisation
  2. Click a ticket to view details, assign it to a technician, change priority or status
  3. Add internal notes (visible only to admin/technicians) or public comments
  4. Set resolution notes when resolving a ticket

Managing Employees

  1. Go to Employees in the sidebar
  2. Edit any employee to change their role:
    • employee — can browse, book, and submit tickets
    • technician — same as employee (future: assigned to tickets)
    • manager — full admin access
    • admin — full admin access
  3. Deactivate employees by unchecking "Active" — they won't be able to log in

Adding Locations

When registering an asset, you can add new locations on the fly using the "Add a Location" form at the bottom of the page. Use short codes like W-1300, DC-01, STORAGE-A.
